VIN Decoder:

Невеликий SPA-застосунок для розшифровки VIN-кодів автомобілів через API NHTSA.

Що вміє:

вводиш VIN → отримуєш характеристики авто;
показуються тільки ті поля, де є значення;
зберігаються останні 3 запити (localStorage);
можна повторно вибрати VIN зі списку;
окрема сторінка зі всіма змінними;
сторінка з детальним описом кожної змінної;

Технології:

React;
React Router;
CSS;
Vite;

API:

Використовується відкрите API NHTSA:https://vpic.nhtsa.dot.gov/api/

Структура репозиторію:

В репозиторії є дві гілки:

main — тільки README;
master — весь вихідний код застосунку;

Як запустити:

git clone <посилання на репозиторій>;
cd <назва папки>;
git checkout master або git switch master;
npm install;
npm run dev;


Демо:https://test-task-rho-six.vercel.app/;

Особливості:

базова валідація VIN;
обробка помилок з API;
HTML-опис змінних рендериться через dangerouslySetInnerHTML;
верстка на чистому CSS;

Примітка:
Спочатку використовував Tailwind для швидшої верстки, але потім переписав все на звичайний CSS відповідно до умов завдання;
